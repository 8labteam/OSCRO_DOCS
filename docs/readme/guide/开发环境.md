# 开发环境使用

开发环境主要是为了用于平台上启动前的适配工作，非长期运行部分，平台上的提供的基础环境，为了统一技术基线版本  
可创建角色： PE， DE   
仅自己创建的dev scene 才有操作权限

开发和使用参考 [开发环境](../tutorials/开发环境.md)
